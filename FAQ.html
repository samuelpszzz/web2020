Página web Desarrollada por el alumno Samuel Pimentel esta es la entrega del proyecto final de diseño páginas web de 2do de informática profesor Sebastián cuadro. El proyecto debe implementar la utilización de las tags estudiadas en el transcurso del año.


Toda la página web esta desarrolla con HTML5 y CSS3, en este caso no se utilizado ninguna librería de JS o la estructuración pre armada de ningún framework.


Todo el contenido audiovisual utilizado en el proyecto no tiene finalidad de monetización o divulgación por el internet ya que el proyecto de desarrolla en un entorno LAN.

La página brinda noticias sobre juegos o noticias referentes al mundo gamer. Pero también hay una sección de programas, donde se realiza una breve descripción y enlaces de descarga.
